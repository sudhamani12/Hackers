The <h1> to <h6> Elements with example

The <h1> to <h6> elements are HTML tags used to define headings for a webpage. These elements are used to indicate the different levels of importance of the text contained within them.

The <h1> element is used for the most important heading, <h2> for the second most important heading, and so on, with <h6> being used for the least important heading.

Example:

<h1>Main heading</h1> <h2>Secondary heading</h2> <h3>Tertiary heading</h3> <h4>Quaternary heading</h4> <h5>Quinary heading</h5> <h6>Senary heading</h6>

In the above example, different levels of headings are defined using the <h1> to <h6> elements. The text contained within each set of <h1> to <h6> tags will be displayed as a separate block of text on the webpage with different levels of font size and weight.

These headings are used to organize the content of the webpage and to make it easier for users to scan and understand the content of the webpage. They also help search engines to understand the structure and the importance of the content on the webpage.

Note: These heading elements are used for SEO and accessibility, it's recommended to use them in a hierarchical order and not to skip any heading level.

The <header> Element with an example

The <header> element is a HTML5 tag used to define a container for introductory content or a set of navigational links. The content within the <header> element typically includes the site logo, site title, site slogan or site header, and site navigation.

Example:

<header> <h1>My Website</h1> <nav> <ul> <li><a href="#">Home</a></li> <li><a href="#">About</a></li> <li><a href="#">Contact</a></li> </ul> </nav> </header>

In the above example, a header is defined using the <header> element. The <header> element contains the site title "My Website" and a navigation menu.

It's usually placed at the top of a webpage and it can be used once per page or document, and it commonly contains the site branding and site navigation.

Note: <header> element is useful to group introductory content or set of navigational links, it can be nested in other elements such as <body> or <article>.

The <nav> Element with an example

The <nav> element is a HTML5 tag used to define a section of a webpage that contains navigation links. The links within the <nav> element are intended to allow the user to navigate the website.
Example:

<nav> <ul> <li><a href="#home">Home</a></li> <li><a href="#about">About</a></li> <li><a href="#services">Services</a></li> <li><a href="#contact">Contact</a></li> </ul> </nav>

In the above example, a navigation menu is defined using the <nav> element. The <nav> element contains a list of links to different sections of the website, such as "Home", "About", "Services", and "Contact".

It's commonly placed in the header or footer of a webpage and it can be used multiple times per page or document, and it commonly contains the site navigation.

Note: <nav> element is useful to group a set of navigation links, it can be nested in other elements such as <header> or <footer>.

In the above example, the main content of the webpage is defined using the <main> element. The <main> element contains the title "Welcome to my website" and the main information about the website, such as a brief description of the website, information about the developer and the services offered.

It's commonly placed in the body of the webpage, it can be used once per page or document and it's meant to provide the main content of the webpage.

Note: <main> element is useful to group the main content of a webpage, it can be nested in other elements such as <body> or <article>.

Unordered Lists with an example

An unordered list in HTML is created using the <ul> element. The <ul> element is used to define an unordered list, and each list item is defined using the <li> element. The items in an unordered list are typically displayed with bullet points or dots.

Definition Lists with an example

A definition list in HTML is created using the <dl> element. The <dl> element is used to define a definition list, and each definition term is defined using the <dt> element and each definition description is defined using the <dd> element.

The <dt> element represents the term or name being defined, and the <dd> element represents the description or definition of the term.

Example:

<dl> <dt>HTML</dt> <dd>Hypertext Markup Language, a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.</dd> <dt>CSS</dt> <dd>Cascading Style Sheets, a style sheet language used for describing the presentation of a document written in a markup language.</dd> <dt>JavaScript</dt> <dd>A programming language commonly used to create interactive and dynamic websites.</dd> </dl>

In the above example, a definition list is defined using the <dl> element. Each definition term is defined using the <dt> element and each definition description is defined using the <dd> element. The terms in this example are "HTML", "CSS" and "JavaScript" and the definitions are the explanation of each of them.

Note: <dl> element is used to create a definition list, <dt> element is used to define the term, and <dd> element is used to define the description, it can be nested in other elements such as <body> or <section>.

Nesting Lists with an example

Lists can be nested in HTML by placing one list inside another list element. This allows for the creation of more complex and organized lists.

Example:

<ul> <li>Fruits <ul> <li>Apples</li> <li>Bananas</li> <li>Oranges</li> </ul> </li> <li>Vegetables <ul> <li>Carrots</li> <li>Potatoes</li> <li>Tomatoes</li> </ul> </li> </ul>

In the above example, an unordered list is defined using the <ul> element. The list items are defined using the <li> elements. The first-level list items are "Fruits" and "Vegetables". The second-level lists are created by nesting another <ul> element inside the <li> element of the first-level list. The second-level list items are different types of fruits and vegetables.

Note: Lists can be nested using any type of list element such as <ol> or <dl>, the nesting can be made as deep as needed to achieve the desired structure.

Styling Lists with CSS with an example

Lists can be styled using CSS by applying styles to the list elements such as <ul>, <ol> and <li>.

Example:

ul { list-style-type: square; } li { font-size: 18px; color: blue; margin-bottom: 10px; }

In the above example, the list-style-type property is used to change the bullet points of the unordered list to squares. The font-size, color and margin-bottom properties are used to change the font size, color, and the margin of the list items.

You can also use CSS class to apply the styles to specific lists or list items, for example:

<ul class="fruits-list"> <li>Apples</li> <li>Bananas</li> <li>Oranges</li> </ul>

.fruits-list { list-style-type: disc; }

This way you can apply styles to specific lists while keeping the default styles for other lists.

Note: There are many different properties you can use to style lists, such as list-style-image, list-style-position, list-style-type and line-height which allow you to control the appearance of lists.
